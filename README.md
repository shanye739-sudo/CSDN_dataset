📦 CSDN_dataset
📌 Overview

This repository contains a large-scale technical blog dataset collected from the CSDN platform. The dataset spans four years (2019, 2021, 2023, 2025) and reflects the evolution of software engineering and emerging technologies in the developer community.

📊 Dataset Description

The dataset includes rich metadata and full-text content of technical blog articles. Each record contains:

Title
Publication time
Views
Likes
Comments
Favorites
Full article content
Source URL
Technology category (channel)
🧠 Coverage

The dataset covers multiple technical domains, including:

Artificial Intelligence (AI / LLM / AIGC)
Big Data & Distributed Systems
Cloud Computing & DevOps
Backend & Frontend Development
Databases & Data Engineering
Software Engineering & Algorithms

🚀 Applications

This dataset is suitable for:

Recommendation systems
Graph Neural Networks (GNN / HGCN)
Topic modeling and clustering
Technology trend analysis
Knowledge graph construction
